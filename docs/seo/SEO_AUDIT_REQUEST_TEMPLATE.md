# SEO総合チェック依頼テンプレート（静的書き出し向け）

## 依頼内容

このプロジェクト全体について、SEO観点で総合チェックをしてください。

## 前提

- Next.js を使用
- 静的書き出しで運用する可能性がある
- 実装済みのコード・設定・ページ構成を確認したうえで判断すること
- 一般論ではなく、このプロジェクトに対する具体的な指摘を行うこと

## 必須チェック対象ページ

- `/`

## 確認してほしい内容

### 1. メタ情報

- 各ページの `title`
- `description`
- `metadata`
- OGP / Twitter Card
- canonical
- robots 関連

### 2. 見出し構造

- `h1` が適切に存在するか
- `h2` 以降の階層が自然か

### 3. HTML構造・クローラビリティ

- `main`, `header`, `nav`, `footer`, `section` などの使い方
- クローラーに伝わりにくい実装がないか

### 4. 内部リンク・URL設計

- 主要ページへの導線
- URLがわかりやすいか

### 5. 画像SEO

- `alt` の不足
- 意味のある画像の扱い

### 6. 構造化データ

- 導入すべき schema.org の種類
- 優先度の高いもの

### 7. インデックス制御

- `sitemap.xml`
- `robots.txt`
- noindex が必要なページの有無

### 8. コンテンツSEO

- 情報量が不足していそうなページ
- タイトルや説明文が抽象的すぎる箇所

### 9. 技術SEO

- JavaScript依存が強すぎる箇所
- パフォーマンス面の懸念

## 出力形式

1. **総評**
2. **優先度：高**
3. **優先度：中**
4. **優先度：低**
5. **ページ別チェック**
6. **不足しているSEO施策一覧**
7. **修正に入る場合のおすすめ順**
